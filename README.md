# 🐦 AI Agent - Flappy Bird (Deep Q-Learning)

## 📝 Giới thiệu
Dự án cá nhân ứng dụng thuật toán Học tăng cường (Reinforcement Learning - Deep Q-Network) để huấn luyện AI tự động ra quyết định và chơi game Flappy Bird. 

Dự án không sử dụng luật if-else khô khan mà để AI tự tương tác với môi trường game, tự học từ những lần rơi và tối ưu hóa chuỗi hành động để sinh tồn.

## 🚀 Công nghệ sử dụng
* **Ngôn ngữ:** Python
* **Machine Learning:** PyTorch (Xây dựng Deep Neural Network)
* **Môi trường Game:** Gymnasium, Flappy-Bird-Gymnasium
